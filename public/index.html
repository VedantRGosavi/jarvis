Getting Started with FridayAI

Welcome to FridayAI! Choose your platform guide below:

Windows Installation Guide

  1. Download the FridayAI installer from our website
  2. Run the installer and follow the on-screen instructions
  3. Launch FridayAI from your Start menu or desktop shortcut
  4. Log in with your account credentials
  5. Select your game from the dropdown menu
  6. Launch your game and press Ctrl + Shift + F to open the overlay

Note: For best performance, run both FridayAI and your game as administrator.

macOS Installation Guide

  1. Download the FridayAI .dmg file from our website
  2. Open the .dmg file and drag FridayAI to your Applications folder
  3. Open FridayAI from your Applications folder
  4. When prompted about an unidentified developer, go to System Preferences > Security & Privacy and click "Open Anyway"
  5. Log in with your account credentials
  6. Select your game from the dropdown menu
  7. Launch your game and press Option + Shift + F to open the overlay

Note: You may need to grant additional permissions in System Preferences > Security & Privacy > Accessibility.

System Requirements

Minimum Requirements

  • OS: Windows 10 64-bit
  • CPU: Intel Core i5-4460 / AMD Ryzen 3 1200
  • RAM: 8 GB
  • Storage: 2 GB available space
  • DirectX: Version 11

Recommended Requirements

  • OS: Windows 10/11 64-bit
  • CPU: Intel Core i7-8700K / AMD Ryzen 5 3600
  • RAM: 16 GB
  • Storage: 4 GB SSD
  • DirectX: Version 12

macOS Requirements

Minimum Requirements

  • OS: macOS 11.0 (Big Sur) or later
  • CPU: Intel Core i5 / Apple M1
  • RAM: 8 GB
  • Storage: 2 GB available space
  • Graphics: Metal-compatible GPU

Recommended Requirements

  • OS: macOS 12.0 (Monterey) or later
  • CPU: Intel Core i7 / Apple M1 Pro or better
  • RAM: 16 GB
  • Storage: 4 GB SSD
  • Graphics: Metal-compatible GPU with 4GB+ VRAM
